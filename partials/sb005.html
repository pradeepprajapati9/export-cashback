SB005 Error Fix: Unblock your stuck IGST refund

SB005 aa raha hai aur IGST refund bank me nahi aaya? Paisa dooba nahi — data mismatch se block hai. Yaha wajah aur step-by-step fix.

SB005 error hota kya hai?

SB005 IGST refund rokne wali sabse common error hai. Iska matlab: tumne GST return (GSTR-1) me jo invoice details bhari, wo Shipping Bill (Customs / ICEGATE) ki details se match nahi kar rahi. Do sarkari system confirm nahi kar paate ki ye same invoice hai, isliye system refund hold kar deta hai.

Kyun aata hai? (Common causes)

Mismatch inTypical mistake
Invoice NumberGSTR-1 vs Shipping Bill me alag no. (extra space, prefix, /)
Invoice ValueTaxable / invoice value dono me alag
Invoice DateDate alag entered
GSTIN / PortGalat GSTIN ya port code

Step-by-step fix

  1. Confirm SB005 — ICEGATE pe shipping bill ka refund/scroll status dekho, error code SB005 dikhega.
  2. Mismatch dhundho — GSTR-1 aur Shipping Bill ki invoice line side-by-side rakho; number, value, date compare karo. Aksar ek chhoti galti hoti hai.
  3. Correction route chuno — (a) ICEGATE online correction utility se detail update karo, ya (b) Officer Interface — Customs officer manually verify karke refund sanction karta hai (CBIC ne SB005 ke liye permanent facility di hai).
  4. GST side theek karo (agar zaroori) — galti GST return me thi to GSTR-1 Table 9A me invoice amend karo taaki match ho.
  5. Re-validate & track — correction ke baad system dobara process karta hai, scroll regenerate hota hai. ICEGATE pe sanction hone tak check karte raho.
Time-limit warning: der mat karo. Expired claim window aur late correction hi wo tarika hai jisse exporters ka refund hamesha ke liye doob jata hai. SB005 dikhte hi fix karo.
